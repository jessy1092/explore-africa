衣索比亞：處在改革開放前夜的巨大東非市場
Loading
2020.04.24

衣索比亞這個非洲國家最近的熱度很高，馬雲在2019年11月對衣索比亞進行了訪問，並且阿里巴巴的 eWTP （編註：世界電子貿易平台）正式落地衣索比亞；另外，推特 CEO Jack Dorsey 也幾乎是同時來到了衣索比亞考察。是什麼原因讓這一中一美兩位互聯網大神級的人物同時關注衣索比亞？這裡的互聯網創投生態的發展和現狀到底如何？有哪些大的機會？

非程創新作為首家中非創投孵化平台，一直關注衣索比亞市場，2019年多次來衣索比亞進行各個賽道調研、投資當地初創公司，這是中國創投機構在衣索比亞互聯網科技創業圈的第一筆投資，創造了歷史。那麼接下來我們就來系統性的梳理一下，衣索比亞的國家情況、創投生態、主要賽道和潛在機會。

一、宏觀經濟晴雨表

坐落於非洲之角內陸的東非國家衣索比亞是非洲獨立的象徵。與大部分撒哈拉以南非洲國家不同，除了墨索里尼的義大利（1935-1941）的短暫佔領之外，它從未被殖民過，這也造就了衣索比亞獨特的經濟、政治、社會結構和創投生態。

人口基數大且非常年輕，有極好的移動互聯網禀賦。

衣索比亞人口1.1億，為撒哈拉以南非洲僅次於奈及利亞的人口大國。同時，衣索比亞也是非洲人口增長率最高的國家之一，年平均人口增長率為2.6%，遠高於全球平均水平（1.1%）。據預測，到2050年衣索比亞人口將增長至1.8億；同時，年輕人佔總體人口比例大，其中15歲以下的人口超過五分之二，有超過5400萬活躍的勞動力人口。

龐大的人口基數和充沛的年輕勞動力為衣索比亞經濟增長提供了動力。尤其在移動互聯網領域，這一優勢更為凸顯。目前手機持有人數超過6400萬，其中約1600萬用戶擁有智能手機，位居世界38位。

衣索比亞仍屬低收入國家，但經濟增長遠超非洲同類國家。

儘管自1990年代以來經濟改革取得了進展，衣索比亞仍然是非洲和世界上最貧窮的國家之一，人均GDP為770美元，低於撒哈拉以南非洲地區的平均水平1,574美元；然而，自2000年以來衣索比亞一直是非洲增長最快的經濟體之一。據世界發展指標（World Development Indicator）顯示，在2000年至2018年之間，衣索比亞GDP年增長平均高達6.8％，按GDP總量體量來說，是非洲第七大經濟體。世界銀行預測，2020至2022年間衣索比亞GDP將保持年均8.2%的增速。

在過去二十年中，衣索比亞的經濟增長主要受政府對基礎設施投資的推動。農業轉型、製造業服務業增長也是近年來衣索比亞政府經濟改革的重點目標。當地政府的第二個增長與轉型五年規劃中，計劃到2025年成為低中等收入國家。

二、數字經濟新動能

電信基礎設施發展迅速，互聯網普及率增長空間大。

相較於落後的交通、電力基礎設施，電信基礎設施發展迅速。如前文所提，在衣索比亞，手機持有人數自2008年來高速增長，目前已超過6400萬，智能手機滲透率為15%， 約1600萬用戶，位居世界38位。

(Dara source：Statista  )

自2011年以來，傳音在衣索比亞銷售的每部手機都在其位於亞的斯亞貝巴郊區的工廠組裝。大約700名員工將深圳製造的屏幕，電路板和電池組合在一起，每天生產2,000部智能手機和4000部功能手機。可預見的是，未來衣索比亞的智能手機出貨量和用戶都會進一步增長。

得益於政策改革，衣索比亞電信壟斷結束，外資運營商增值服務市場大。

國有電信公司 Ethio Telecom 曾經是衣索比亞唯一的互聯網服務提供商（ISP），Ethio Telecom 的網絡質量比較一般，而且受政府干預比較嚴重。但阿比政府的一項重大計劃是將國有企業私有化，衣索比亞電信公司、衣索比亞航空公司、衣索比亞電力公司和衣索比亞航運公司的股份將被出售給本國和外國資本家。今年8月，衣索比亞政府針對跨國電信公司發放了2個電信牌照（Vodacom和MTN，另外法國的Orange和阿拉伯聯合酋長國的Etisalat可能是第三塊牌照的競爭者），明年外資電信運營商進場，將帶來千億美金級別的投資和電信運營經驗，衣索比亞電信和ICT市場將迎來巨大發展。

截至2019年1月，首都亞的斯亞貝巴有2G / 3G和4G連接選項，但速率有待提升。數據、語音、短信資費目前相對高昂。以4G數據套餐為例，1G流量的資費是80比爾，相當於15元人民幣左右。

三、創投生態全面解讀

創投生態發展滯後，但變革正在發生。

由於之前較強的監管和封閉的市場，外資受到嚴重限制，經濟活躍程度也受到影響，科技與創投的發展都遠遠落後於南非、奈及利亞、肯亞和埃及。但是，隨著政府新政的施行，更多外國投資者可以參與各個領域。

投資機構極少，本地天使投資網絡出現萌芽

衣索比亞現有投資機構主要針對農產品加工和製造業等行業，針對創投公司幾乎寥寥，和東非鄰國相比，並不是資金追逐的熱門區域。

但同時，以 Bluemoon 孵化器合夥人 Shem 為首的天使投資人，是最早的一批科技創投支持者，衣索比亞天使投資開始的標誌事件是 2017年 Shem 投資了 Ride（25,000美金佔股40%），後來 Ride 成長為衣索比亞第一的打車平台，市場佔有率超過80%，估值也已經翻了100多倍。投資 Ride 為 Shem 帶來了巨大的匯報和名氣。目前 Shem 和其投資合夥人擁有 Ride 的多數股份，高於創始團隊。而且他們正在打造衣索比亞天使投資人網絡，每個月舉辦一次項目路演。

除此之外，2019年新成立的 Kudu Ventures 基金，是一支專注衣索比亞的早期基金，美國背景。目前在和 iCog Labs 合作加速器 Program。

創業公司數量少，競爭小，融資困難

創業公司數量比較少，由於沒有外資進入，所以很少有創業公司走到A輪之後（Ride，ZayRide，Gebeya等）。而且創業公司為了吸引投資，多數同意分紅。衣索比亞目前的創投退出主要方式是分紅。

(衣索比亞主要賽道代表之新創公司。/ Data source：briterbridges)

孵化器、共享空間長期存在，積極貢獻力量

作為創投生態中必不可少的支持力量，孵化器、加速器、共享空間的發展值得關注。衣索比亞的孵化器及共享空間數量並不太多，投資孵化的項目主要從 idea 概念階段開始，面向的對象為大學生。不過，由於衣索比亞的創投生態還處於早期，沒有數量多、質量高的成熟的初創公司，孵化器需要從概念階段就開始對創業者進行輔導和培訓。本地主要孵化器：

IceAddis：於2011年成立，衣索比亞最早的孵化器，當初成立的目的是為衣索比亞的初創公司建立協作網絡。 IceAddis提供辦公空間，為年輕創業者（主要是大學剛畢業的學生）提供輔導和支持，以提供的資源換10%股權，目前共孵化40多家初創團隊。主要收入來源是諮詢服務和活動承辦。

BlueMoon：衣索比亞著名孵化器 BlueMoon 和辦公空間 BlueSpace，2017年成立，主要關注的賽道有農業科技、新興技術、區塊鏈、3D、AI、大數據。從項目的idea階段就開始介入支持，提供10K美元的種子輪資金（10%），對創業者培訓，輔導，6個月後，舉行 pitch Day，邀請各位天使投資人，對優勝者繼續投資50 -200K美元。項目創業者主要為大學畢業生。衣索比亞每年有約50萬大學畢業生，政府提了供低息貸款支持創業。

iCogLabs：iCog Labs是一家位於衣索比亞的研發公司，與國際AI研究團隊合作，為全球客戶提供外包的技術服務。幫助客戶完成了各種項目，核心專長是人工智能，包括基於機器學習的數據分析，計算語言學，計算機視覺，移動機器人和認知機器人，認知架構和人工智能。為國際AI開發團隊提供經驗，並為衣索比亞政府和行業項目提供軟件和硬件開發經驗，是 OpenCog開源AI平台的核心貢獻者。另外也開展加速器項目， 從全國各地篩選項目進入加速器計劃（8月開展），同時也和各個國際機構或跨國公司一起開展創業公司挑戰賽，擴大技術創新的影響力。

四、主要賽道機會及代表公司訪談

◎ Mobile Money賽道

衣索比亞仍然是一個以現金為主導的社會。 2017年，擁有銀行帳戶的成年人比例上升至35％，但根據Findex的數據，同年只有0.3％的成年人擁有移動貨幣帳戶，在東非國家中屬於滯後的水平。

截止到目前2019年底，mobile money 用戶規模在200萬人左右。由於衣索比亞的金融監管非常嚴格，不允許運營商（Ethio Telecom）提供金融服務，只有銀行和金融機構才可以提供金融服務，創業公司想要提供 mobile money 服務，必須跟銀行合作，而且 mobile money 領域還沒有對外資開放。

但是很多政策目前正在鬆動。 

2012年，衣索比亞國家銀行製定了一套監管移動和代理銀行業務的政策，於 2013年初生效。現在通過少數金融機構在各個地區提供移動貨幣服務。M-Birr 在12月獲得監管部門批准後，於3月份正式啟動。 M-Birr 合作五家小額信貸機構，主要側重於農村地區的貧困家庭，聲稱擁有120萬用戶，7000個代理商。

另外，在2019年8月，政府給一家私人公司發放了Leasing牌照（融資租賃）。

目前的 Mobile money 產品有：銀行系的 Mobile money 及 Mobile wallet、私人公司的 Mobile money 

（1）銀行系的 mobile money

存取款通過線下代理，轉賬和充值通過USSD短號，沒有APP

代表產品：CBE-BIRR（衣索比亞商業銀行，國有銀行）、M-Birr

（2）銀行系的 mobile wallet

存取款通過線下代理，轉賬充值通過APP，可以在線支付，有APP

代表產品：Amole（Dashen Bank一家私有銀行的產品）

（3）私人公司的 mobile wallet

存取款通過線下代理，轉賬充值通過USSD短號或者APP，可以進行線上支付，需要接入銀行的mobile money，本質上是聚合支付。代表產品：HelloCash、YenePay。

主要 Mobile money 公司訪談：

BelCash：B2B2C 模式，理想是打造類似微信一樣的超級 App， 目前提供的服務主要有：HelloCash、Hello Airtime、HelloMarket、HelloSolar。

  • HelloCash : 移動支付，agent banking, 聲稱目前有120萬用戶，8000個代理商，6個合作銀行，2個小額信貸銀行，沒有資金池，資金通過銀行運轉
  • Hello Airtime : 有話費池，分發運營商的話費
  • HelloMarket : 有自己的倉庫，目前有15,000小商戶，有線下提貨
  • HelloSolar : 衣索比亞只有30%的人口能用上電，該服務提供家用的離網太陽能組件的租賃購買，12-18個月內還清。

另外，娛樂類產品將於2020年上線。

  • YenePay：移動支付產品。目前旗下有四款產品，通過交易佣金獲得收入。
  • Yenepay Ecommerce - 這是YenePay.com的核心產品，它面向電子商務企業
  • YenePay Mobile : 這適用於移動用戶的移動支付服務
  • YenePay E-invoice : 主要針對希望通過開具發票來收款的企業和個人
  • YenePay Bulk Pay : 它是針對一次向大量人員支付大筆款項的企業和個人
  • 另外，YenePay Remittance正在開發中。

◎ 電信增值服務賽道

2019年衣索比亞電信收入100億比爾，其中電信增值服務收入 5.9%，約為 5.9億比爾，約等於 2500萬美金。由於信用卡普及率低、移動支付發展滯後等原因，目前衣索比亞內容提供商最主要的收款方式是通過電信扣費。因此，CP和SP都有很大機會。

(Photo by Seyi Ariyo on Unsplash)

◎ 出行賽道

首都 700 萬居民至少需要 17,000 輛公共汽車才能滿足需求，政府僅滿足了 69.7％ 的需求，只有4%的人口擁有私人交通工具。 

Uber 目前沒有進入衣索比亞，本地網約車服務 Ride 和 Zayride 興起，填補缺口。目前衣索比亞共有20家左右的打車平台在競爭。首都圈潛在規模一天10萬單，目前的打車平台日單在25000單左右，還有一定的增長空間。

Ride：提供本地出行服務，2014年成立，Uber模式。衣索比亞沒有 uber 和taxify（交通領域暫時沒有向外資開放），因此本地叫車應用佔領了市場。 Ride 目前仍在快速增長，2019年底市場佔有率約80%，目前開始向投資人分紅。同類競品還有 Zayride, 成立於2016年，2019年初完成了A輪融資，目前已經開展送貨服務。

◎ 本地生活賽道

衣索比亞的本地服務發展比較滯後，雖然人口基數龐大，有較大的市場空間，但是網絡接入程度低且網絡狀況不夠穩定，因此沒有湧現出具備一定規模的本地服務公司。不過，在新一輪經濟改革中，政策正在變革，該領域有可能湧現極具競爭力的本地玩家。

Delivery Addis：外賣配送業務。主要流程為用戶下單後，平台向餐廳打電話確認訂單，之後派騎手取餐送餐。盈利模式主要通過收取訂單7%作為佣金，給騎手固定費用。已從美國融資50萬美元。主要面臨的挑戰是衣索比亞政府禁摩的政策，認為該政策存在遊說和鬆動的空間。

TaskMoby：BelCash聯名產品，及時服務平台（on-demand service platform）, 目前在3個國家運營（衣索比亞，坦尚尼亞，烏干達），沒有獲得外部投資，曾收到過創業類獎金。提供的服務主要有：幫助谷歌等大公司收集各項服務提供者的信息並更新，收取諮詢費。黃頁服務，商業模式主要是廣告+和運營商分成用戶電話諮詢。

即時服務平台：目前提供5類服務（清潔、電工、疏通水道等）已運營6個月未來會提供培訓和審核，升級工人為企業客戶，也考慮為他們提供小額貸款。

◎ 開發人員培訓賽道

衣索比亞距離歐洲比較近，文化上容易溝通。而且由於人口基礎大，人力成本低，成為了繼越南後的下一個製造基地。從技術開發方面來說，越來越多的衣索比亞大學生加入學習代碼和編程的行業，進入培訓學校學習後，為國外的公司提供外包技術服務。目前主要的培訓類公司是 Gebeya，在衣索比亞，肯亞，吉布地均有辦公室。 Gebeya 已經完成了A輪融資，在衣索比亞和矽谷註冊，馬上開始下一輪融資。目前提供三項服務：培訓、人才市場、空間。

培訓：4-6個月電腦技術培訓，適用於已經有基礎的人，線下課程，大部分老師是全職。

人才市場：四項服務，G-talent、G-subscription、G-made、G-box

  • G-talent : 提供短期開發人員，Contract
  • G-subscription：每個月僱傭固定幾個小時的開發人員
  • G-made : 僱傭核心開發人員完成產品開發
  • G-box：直接使用已經開發完畢的系

空間：為初創公司提供共享空間和資金，Kukulu（3D 移動遊戲）是目前唯一一家入駐的公司。

◎ 醫療服務賽道

衣索比亞人口眾多，存在大量的農村人口和貧困人口，難以獲得安全的水，住房，衛生設施，食品和衛生服務。衣索比亞估計有 85％ 的農村人口無法獲得醫療保健。在衣索比亞，愛滋病、結核病、瘧疾、呼吸道感染和腹瀉等傳染病仍然是嚴峻的挑戰。

衣索比亞有 1,949個衛生站和 141個衛生中心，這些設施中許多都沒有醫生在場以提供護理。 2012年，政府新建了 13所醫學院，並增加了已經建立的醫學院的入學人數。政府提出，公共部門有 2,152位醫生（每 42,000名患者中大約有一位醫生）。同一項調查顯示，從衣索比亞居留計劃畢業的醫師中，有 73％離開政府公共部門從事商業，或者移民海外以獲得更多收入。

美國衣索比亞醫院於 2017年4月開始建設，計劃於 2019年12月開始運營，政府希望通過公私合營（Public Private Partnership）的機制鼓勵外國對該行業的投資。衣索比亞政府目前正在該國各地建立四個地區血庫辦事處和十二個小型血庫。

目前醫療服務領域主要的初創公司是 Orbit Health，提供 to B 服務，即向醫院提供醫療軟件系統，比如門診預約系統、電子病歷系統、健康數據管理、健康諮詢等精品醫療健康服務。該公司通過創​​新的技術解決方案，可以將等待時間和患者周轉時間縮短 55％ 以上。

◎ 農業科技賽道

由於農業在衣索比亞基礎經濟中的重要性，農業科技（Agritech）行業內興起很多創業公司，著名的孵化器 BlueMoon 針對這個賽道有重點投入。

◎ 能源電力Cleantech賽道

能源電力基礎設施的缺乏，導致衣索比亞有近 70%的人無法獲得低價穩定的電力服務。所以清潔能源（cleantech）賽道也興起了創業公司。

五、非程結語

衣索比亞與曾經的中國非常相似，龐大而年輕的人口、管制嚴格相對孤立的市場，逐漸開放和變革的政府、基建和製造業方面的明顯進展等等。數字經濟的興起使得衣索比亞在金融、本地服務、電商、零售、物流、農業科技方面，蘊藏著巨大的機會和潛力。由於缺乏外國競爭者，本地玩家有較長的時間窗口完成擴張，不過可能由於沒有外國科技公司的培養，本地創業者會缺乏一定的國際化戰略視野。 

在早期投資方面，目前處於比較早的入場階段，外國投資者很少，本國投資人不夠關注科技行業，導致初創公司種子輪資金很難募集，項目的估值都較低。對於中國投資人來說，可以多關注能在短期內產生收入，回流現金的賽道，如果項目本身還有美元儲備，是最佳的選擇。通過股利分配回收投資也是一個不錯的退出途徑。小額貸、博彩、出行等行業也可以重點關注。另外，2020年衣索比亞會舉行大選，可能會存在一定的不穩定性和風險。

對於創業者來說，選擇進入衣索比亞市場，需要對賽道相應監管政策（尤其是對外資的限制）和退出渠道做好充分研究和準備。

對於非程創新來說，我們的策略是在市場開放和起飛之前，就做好早期佈局，積累政策理解與政府關係，本地合作夥伴，以及賽道的理解和認知。最為重要的是衣索比亞本土頭部的互聯網創業人才和團隊。非程創新通過早期投資和孵化的方式，將會最大程度地將衣索比亞本地人才聚合起來，不僅對衣索比亞生態提供價值，同時也為日後中國資本和創業者提供對接服務。
